# Diamond / Diamant

**Niveau :** Moyen

## Énoncé

L'énoncé de ce kata est court : 

> Étant donné une lettre, imprimez un diamant commençant par 'A' avec la lettre fournie au point le plus large.

**Par exemple** : Pour la lettre `E`

```
    A
   B B
  C   C
 D     D
E       E
 D     D
  C   C
   B B
    A
```

**Par exemple** : Pour la lettre `C`

```
  A
 B B
C   C
 B B
  A
```


Have fun :D
